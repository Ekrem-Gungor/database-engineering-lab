# Database Engineering Lab

Bu repository, veritabanı mühendisliği becerilerimi geliştirmek için tasarlanmış kişisel bir mühendislik laboratuvarıdır.

Bu projenin amacı, aşağıdaki gibi gerçek dünya veritabanı problemlerini keşfetmektir:

- SQL analitiği
- Sorgu optimizasyonu
- İndeksleme stratejileri
- İşlemler ve eşzamanlılık
- Bölümlendirme
- Önbellekleme
- Veri mühendisliği temelleri

## STeknoloji Yığını

- PostgreSQL
- Docker
- pgAdmin
- Redis (gelecek aşamalar)

## Versiyonlama

Bu proje, semantik versiyonlama yöntemini takip eder.

v0.0.0 → İlk laboratuvar kurulumu

v0.4.0 → Operasyonlar ve güvenilirlik
v1.0.0 → Tamamlanmış veritabanı mühendisliği laboratuvarı

## Alan adı

Proje, gerçek dünya veritabanı senaryolarını simüle etmek için bir **e-ticaret veri modeli** kullanır.
